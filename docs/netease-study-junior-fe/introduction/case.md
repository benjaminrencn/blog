## 开发者工具

* Chrome 开发者工具打开方式
  * `command` + `option` + `I` `F12` `Ctrl` + `Shift` + `I`
  * 设置 > 更多工具 > 开发者工具
* Elements
  * <img src="./assets/case-elements.png" alt="开发者工具 Elements">
  * 查看 DOM 树结构，修改样式、类。查看盒模型、样式计算结果、DOM属性
* Sources
  * <img src="./assets/case-sources.png" alt="开发者工具 Sources">
  * 查看脚本，打断点调试，查看对应作用域的变量、调用栈
